# HyperLogLog
Proyecto 1 para curso Tópicos en Manejos de Grandes Volúmenes de Datos.

## Necesario
1. Linux
2. cmake (>= 3.1)
3. g++ (>= 11.0)
4. C++11


## Instructiones
```bin/bash
git clone https://github.com/bletelier/HyperLogLog.git
cd HyperLogLog
chmod +x build.sh
./build.sh
./compile.sh
./run.sh
```
