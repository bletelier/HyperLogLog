# HyperLogLog
Proyecto 1 para curso Tópicos en Manejos de Grandes Volúmenes de Datos.

## Necesario
1. Linux
2. cmake (>= 3.1)
3. g++ (>= 11.0)
4. C++11


## Instructiones
```bin/bash
git clone https://github.com/bletelier/HyperLogLog.git
cd HyperLogLog
chmod +x build.sh
./build.sh
./compile.sh
./run.sh
```
## Como ejecutar **kmers** con un K específico (ej: K = 25)
```bin/bash
./build/main 25
```

## Como ejecutar **minimizers** con un K y W específico (ej: K = 25 W = 100)
```bin/bash
./build/main 25 100
```
